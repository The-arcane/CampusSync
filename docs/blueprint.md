# **App Name**: CampusSync

## Core Features:

- Role-Based Access: Secure access control based on roles: Super Admin, Admin, Teacher, Security/Staff, and Parent, managed through Supabase Auth.
- Student & Staff Management: Manage student admissions, class assignments, parent linking, and staff records (teachers and non-teaching staff).
- QR Attendance System: Log student and staff attendance via QR codes, tracking entry and exit times, calculating late arrivals, and automating monthly attendance reports.
- Automated Payroll: Automatically calculate monthly payroll based on defined rules (late arrivals, leaves) with clear deduction policies, editable only by Super Admin.
- Academic Records: Teachers can upload exam marks, remarks, and academic reports, with parents able to view child profiles, attendance, results, and download report cards.
- Analytics Dashboard: Real-time dashboards for each role, showing attendance trends, salary summaries, and student performance.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust, knowledge, and professionalism, reflecting the academic environment.
- Background color: Light grey (#F0F2F5), a desaturated hue from the primary color for a clean, modern, and calm user interface.
- Accent color: Warm orange (#FF9800), an analogous color to indigo, will provide contrast and highlight key interactive elements and calls to action.
- Headline font: 'Space Grotesk', a sans-serif with a modern techy look suitable for headings; Body font: 'Inter' a grotesque-style sans-serif font suitable for body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a consistent set of minimalist icons to represent different modules and actions, ensuring clarity and ease of navigation.
- Implement a responsive grid layout that adapts seamlessly to different screen sizes and devices, ensuring a consistent user experience.
